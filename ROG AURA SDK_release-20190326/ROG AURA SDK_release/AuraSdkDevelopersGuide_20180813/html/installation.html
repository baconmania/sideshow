Installation

Before programming with Aura SDK 3.0, you should install the SDK package and HAL packages for all devices which you want to be supported.

Versions 1.02.01 and after

Start from version 1.02.01, the installation process of Aura SDK package and all HAL packages has been integrated with one single setup program.

  1. Unzip the released file, and you will get sdk.exe and a folder named "aac":
  2. Run sdk.exe and all necessary files will be installed and registered.

Versions before 1.02.01

You have to install the SDK package and HAL packages for all devices which you want to be supported manually. Since the SDK will search for supported HALâ€™s in run-time, so the installation order is not important.

The installation process of SDK & HAL packages is simpile: just double-click on the Setup.exe icon of each package, then follow the instructions of the setup wizard, and everything is done.
